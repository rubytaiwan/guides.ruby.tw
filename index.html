如何幫忙翻譯文件？

如果您有興趣加入翻譯計畫，歡迎至以下的 GitHub 專案的 Wiki 頁面登記你的大名。如果您是 GitHub 使用者，您可以 fork 這個專案。完成之後送 pull request 即可。或是您可以來信給 ihower 說明你想加入維護這個專案成為 Collaborator。如果您不會使用 Git，請用原始(raw)格式進行翻譯，例如 About Ruby 這頁，然後將翻譯的結果 email 給 ihower。
